# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## Features
- Real-time collaboration with live cursors
- AI Diagram Generator using Claude API  
- Handwriting to Text using GPT-4o Vision
- Voice to Diagram using Whisper API
- Share boards via link or QR code
- Export boards as PNG
- Works on mobile and desktop

## Tech Stack
Frontend: React, Vite, TailwindCSS, Fabric.js, Firebase
Backend: Node.js, Express
AI: Claude API, OpenAI GPT-4o, Whisper
Database: Firebase Firestore, Firebase Realtime Database
Deployment: Vercel (frontend), Railway (backend)
